Information regarding Backup


Backup contains data of all telephony applications (Baang, Sawaal, Polly, and Rahemma) that were deployed, using Wateen server, in three launches of 2015, 2016 and 2017.


Backup Directory Structure


The backup data is in three parts, pertaining to different launches, sub-directories marked by launch year. Data means Database, htdocs, and telephony server-side scripts and configuration files of the particular launch.


Backup_Polly_Baang_Sawaal_2015_2016_2017 — root directory of backup.

2015 Launch
— data of First Launch.

2016 Launch
— data of Second Launch.

2017 Launch
— data of Third Launch.

Applications Deployed

Note :- Applications Deployed


  • Polly, and Baang were deployed in all three launches ( 2015 , 2016 and 2017 ).
  • Sawaal was deployed in launches of 2016 and 2017 .
  • Surveys and Rahemma were deployed in 2017 only.
  • HelloRozgar was deployed in 2015, but unlike other applications it was not stopped/monitored after that, it continued to run passively uptill now.
